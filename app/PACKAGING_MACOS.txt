macOS packaging (must run on a Mac)
====================================

1) Terminal:
     cd app
     bash build_macos.sh
   Or from repo root double-click: 一键打包_macOS.command

2) Outputs (under app/dist/)
   - TidanMgr.app                 Double-click to run (data in ~/Library/Application Support/TidanMgr/)
   - PortableStart.command        Portable mode: data in dist/TidanMgrData/ (copy together with .app)
   - run_TidanMgr.sh              Run from Terminal with logs (portable data)
   - TidanMgr-macos-portable.zip  Zip of the three items above (ASCII filename, for distribution)

3) First launch "cannot be opened": System Settings -> Privacy & Security -> Open Anyway; or right-click .app -> Open.

4) Put template.xlsx in the app/ folder before packaging so export bundles the template.
