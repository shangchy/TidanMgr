# 提单管理系统详细设计文档

## 1. 文档信息
- 文档名称：提单管理系统详细设计文档
- 版本：V1.0
- 编写日期：2026-04-18
- 关联文档：`docs/需求规格说明书.md`
- 技术栈：Python + PySide6 + openpyxl

## 2. 总体设计
- 架构分层：View（PySide6） / State（内存+JSON） / Logic（校验联动导出）
- 运行平台：macOS + Windows
- 打包方式：PyInstaller（`.app` / `.exe`）
- 数据存储：`app/data.json`、`app/theme.json`

## 3. 核心模块
- 主题模块：深浅色切换与持久化
- 列表模块：行内编辑、首列勾选、表头全选/反选、操作列删除
- 任务名模块：正向拼接与反向解析
- 省市模块：多选弹窗、互斥编辑锁、排除字段联动
- 校验模块：字段校验与跨字段校验
- 导出模块：基于 `template.xlsx` 第2行开始写入

## 4. 数据模型
```json
{
  "task_name": "",
  "type_code": "",
  "operator_code": "",
  "industry_code": "",
  "url": "",
  "quantity": "",
  "duration": "",
  "age_max": "",
  "age_min": "",
  "pv": "",
  "province": "",
  "exclude_province": "",
  "city": "",
  "exclude_city": "",
  "checked": false
}
```

## 5. 字段与联动规则
- 码值映射：类型/运营商存码值，显示名称，任务名使用码值
- 任务名格式：
  - `客户名 + 地域片段 + 运营商码值 + 类型码值 + 行业编码 + "-" + 产品名`
- 地域片段：
  - 单值显示地名
  - 多值显示“几省/几市”
  - 空值显示“全国”
- 双向联动：
  - 修改字段自动重建任务名
  - 修改任务名可回填运营商/类型/行业/地域

## 6. 校验设计
- 字段校验：
  - `quantity`、`age_max`、`age_min`、`pv` 仅允许数字
  - `quantity > 0`
- 跨字段校验：
  - `age_max >= age_min`
- 错误处理：
  - 非法输入回滚
  - 错误单元格红色高亮

## 7. 互斥与锁定
- 正向字段：`province` / `city`
- 反向字段：`exclude_province` / `exclude_city`
- 锁定规则：
  - 正向有值 -> 反向不可编辑
  - 反向有值 -> 正向不可编辑
  - `province` 与 `city` 二选一

## 8. 导出设计（模板模式）
- 模板文件：`template.xlsx`
- 起始写入行：第2行
- 写入方式：仅改单元格值，不改模板结构/样式/列宽
- URL策略：由“是否打印URL”控制是否写入 URL 列
- 文件名规则：`客户名_YYYYMMDD.xlsx`
- 列映射：
  - A:任务名 B:类型 C:运营商 D:行业编码 E:URL
  - F:数量 G:时长 H:年龄上限 I:年龄下限 J:pv
  - K:省份 L:排除省份 M:地市 N:排除地市

## 9. 交互设计
- 首列：点击单元格即可切换勾选状态
- 表头首列：对当前可见数据执行全选/取消全选
- 新增行：
  - 按钮新增
  - 双击最后一行新增
  - 下拉字段默认为未选
- 删除：
  - 单行删除按钮
  - 批量删除选中
  - 删除已保存数据需要二次确认

## 10. 非功能与发布
- 全离线可用
- 代码目录：`app/`
- 文档目录：`docs/`
- 打包脚本：
  - `app/build_macos.sh`
  - `app/build_windows.ps1`
